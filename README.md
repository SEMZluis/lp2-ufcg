## 📖 Sobre a disciplina 
Segundo o [Projeto Pedagógico de 2023](https://cgcc.ufcg.edu.br/docs/ppc-2023.pdf), a ementa da disciplina é composta por:
- Fundamentos de linguagens de programação (neste caso, Java);
- Estruturas, tipos e controle de fluxo;
- Abstração, encapsulamento e escopo;
- Paradigmas: imperativo, funcional, lógico, OO e concorrente;
- Introdução a novos paradigmas.
